# 作者使用手册（最短路径）

## 1) 进入写作

- 默认用“写作模式”：只保留正文 + 章节，减少干扰。
- 右上角可切“工作台模式”：查看 evidence、动作 payload、检索细节。
- 工作台模式顶部可按需勾选面板（Prompt/结构化大纲/检索快照），默认只开“结构化大纲”。

## 2) 呼出助手

- 点击右下角“助手”浮动按钮，或快捷键 `Ctrl/Cmd + Shift + A`。
- 抽屉默认关闭，避免长期占屏。

## 3) 最短写作闭环

- Ghost Text：正文停顿约 1.5 秒后会自动出现浅灰补全建议。
  - `Tab`：接受整段建议
  - `Ctrl+→`：只接受一个词
  - 继续输入：建议会自动刷新或消失
- 选区润色/扩写：先选中文本，点“润色选中/扩写选中”。
  返回结果不会直接覆盖正文，而是以行内 Diff 形式显示：
  - 绿色高亮：AI 建议新增
  - 红色删除线：原文将被替换
  - 每条修改都可单独“接受/忽略”
- 助手回复写回正文：
  - “插入助手回复”
  - “替换选中为助手回复”

## 4) 动作应用与撤销

- 动作提议里“应用并记录”会立刻写入设定/卡片并记录审计日志。
- 已应用动作可点“撤销应用”回退。

## 5) 安全感机制

- 零感保存：正文会自动保存并显示状态（等待中/保存中/已保存）。
- 本地崩溃恢复：未落库正文会写入 `localStorage` 快照。
  刷新或异常后若快照更新，会自动恢复并提示。

## 6) 降级时的界面反馈

- 写作模式仅显示“已降级（不中断写作）”小提示，不打断写作流。
- 工作台模式才展开降级原因与 evidence 细节。

## 7) 设置面板分层

- “写作设置”按三段折叠：基础写作（默认展开）/ 模型与检索 / 高级调优。
- 新项目建议先改“基础写作”，其余保持默认，写通闭环后再细调。
- “模型与检索 / 高级调优”每个字段都附了简短说明，不懂术语时可先看说明再改。
- 字段右上角 `?` 可悬浮查看说明，默认界面更简洁。
